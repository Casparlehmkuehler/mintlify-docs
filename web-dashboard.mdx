## Overview

The Lyceum Cloud web dashboard provides a comprehensive interface for platform management, monitoring, and administration. Access your dashboard at [dashboard.lyceum.technology](https://dashboard.lyceum.technology).

## Core Features

### Dashboard Home
- **Overview Statistics**: Quick stats on recent activity, credit usage, and active runs
- **Recent Activity**: Timeline of recent executions and operations
- **Quick Actions**: Shortcuts to common tasks like starting new runs or uploading data

### Runs Management
- **Execution Overview**: Horizontal scrollable cards showing all runs
- **Detailed Information**: Side panel with comprehensive run details
- **Mobile Responsive**: Optimized layout for mobile devices with full-screen details

### Billing & Credits
- **Credit Balance**: Current available credits and usage history
- **Transaction History**: Detailed billing records with timestamps
- **Usage Analytics**: Breakdown of resource consumption by time period
- **Payment Management**: Integration with Stripe for credit purchases

## Navigation

### Desktop Interface
- **Expandable Sidebar**: Toggle between collapsed (icons only) and expanded views
- **Active States**: Visual highlighting of current page
- **Smooth Transitions**: Fluid animations throughout the interface

### Mobile Interface
- **Slide-out Navigation**: Touch-friendly drawer menu on mobile devices
- **Full-screen Details**: Optimized viewing for execution details
- **Responsive Design**: Seamless experience across all screen sizes

### Detailed Run Information
Click any execution card to view comprehensive details in the side panel:

#### Execution Logs
- **Real-time Output**: Live streaming of execution logs
- **Error Tracking**: Highlighted errors and stack traces
- **Search & Filter**: Find specific log entries quickly

#### Resource Usage
- **CPU/Memory Monitoring**: Real-time resource consumption
- **Cost Tracking**: Per-execution cost breakdown
- **Duration Analysis**: Execution time and performance metrics

#### Configuration Details
- **Machine Type**: Hardware specifications used
- **Environment**: Runtime configuration and dependencies
- **Input Files**: List of uploaded files and parameters

### Execution Actions
- **Abort Running Jobs**: Stop executions in progress
- **Restart Failed Jobs**: Re-run executions with same configuration
- **Download Results**: Retrieve output files and logs
- **Share Execution**: Generate shareable links for collaboration

## Technical Specifications

### Framework & Technology
- **Frontend**: React 18 with TypeScript
- **Styling**: TailwindCSS with responsive design
- **Animations**: Framer Motion for smooth transitions
- **Routing**: React Router for single-page application navigation
- **Icons**: Lucide React icon library

### Performance
- **Build Tool**: Vite for fast development and optimized builds
- **Lazy Loading**: Components loaded on demand
- **Caching**: Intelligent data caching for improved performance
- **PWA Support**: Progressive web app capabilities

### Security
- **JWT Authentication**: Secure token-based authentication
- **HTTPS Encryption**: All data transmitted securely
- **Session Management**: Automatic session timeout and renewal
- **CSRF Protection**: Built-in cross-site request forgery prevention

## Best Practices

### Dashboard Usage
- **Regular Monitoring**: Check execution status and resource usage regularly
- **Cost Management**: Set up alerts to monitor spending
- **Security**: Use strong passwords and enable 2FA
- **API Keys**: Rotate keys regularly and use minimal permissions

### Performance Tips
- **Browser Cache**: Keep browser cache enabled for better performance
- **Multiple Tabs**: Use multiple browser tabs for different workflows
- **Bookmark Shortcuts**: Save direct links to frequently used sections

## Troubleshooting

### Common Issues
- **Login Problems**: Clear browser cache and cookies
- **Display Issues**: Try different browsers or disable ad blockers
- **Slow Loading**: Check internet connection and server status
- **API Key Issues**: Verify key permissions and expiration dates

### Browser Compatibility
- **Supported Browsers**: Chrome, Firefox, Safari, Edge (latest versions)
- **JavaScript Required**: Dashboard requires JavaScript to be enabled
- **Cookies**: Third-party cookies must be allowed for authentication
