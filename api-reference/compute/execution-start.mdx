## POST /compute/execution/python/start

Start a new Python code execution with specified hardware resources.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token or API key authentication
</ParamField>

### Body

<ParamField body="code" type="string" required>
  Python code to execute
</ParamField>

<ParamField body="requirements" type="array">
  List of pip packages to install
</ParamField>

<ParamField body="machine_type" type="string">
  Hardware profile (e.g., "cpu.small", "gpu.a100.40gb")
</ParamField>

<ParamField body="timeout_seconds" type="integer">
  Maximum execution time in seconds
</ParamField>

<ParamField body="environment_variables" type="object">
  Environment variables to set
</ParamField>

### Response

<ResponseField name="execution_id" type="string">
  Unique identifier for the execution
</ResponseField>

<ResponseField name="stream_url" type="string">
  WebSocket URL for real-time output streaming
</ResponseField>

<ResponseField name="status" type="string">
  Initial execution status
</ResponseField>

<ResponseField name="estimated_cost" type="number">
  Estimated cost in USD
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.lyceum.cloud/api/v2/external/compute/execution/python/start \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "import numpy as np\nprint(np.random.rand(5))",
    "requirements": ["numpy==1.24.0"],
    "machine_type": "cpu.small",
    "timeout_seconds": 300
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.lyceum.cloud/api/v2/external/compute/execution/python/start",
    headers={"Authorization": "Bearer YOUR_TOKEN"},
    json={
        "code": "import numpy as np\nprint(np.random.rand(5))",
        "requirements": ["numpy==1.24.0"],
        "machine_type": "cpu.small",
        "timeout_seconds": 300
    }
)
```

</RequestExample>

<ResponseExample>

</ResponseExample>
