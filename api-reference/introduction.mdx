## Welcome to Lyceum API

The Lyceum API provides comprehensive access to cloud compute resources, AI model inference, and storage capabilities.

## Base URL

```
https://api.lyceum.cloud/api/v2/external
```

## Authentication

All API endpoints require authentication using either:

### API Key Authentication
Include your API key in the request header:
```
X-API-Key: lyceum_xxx
```

### Bearer Token Authentication
Exchange your API key for a JWT token and include it in the Authorization header:
```
Authorization: Bearer <token>
```

## Response Format

All API responses follow a consistent JSON format:

```json
{
  "data": {...},
  "status": "success",
  "timestamp": "2024-01-01T00:00:00Z"
}
```

## Error Handling

Error responses include appropriate HTTP status codes and descriptive messages:

```json
{
  "error": {
    "code": "RESOURCE_NOT_FOUND",
    "message": "The requested resource was not found",
    "details": {...}
  }
}
```
