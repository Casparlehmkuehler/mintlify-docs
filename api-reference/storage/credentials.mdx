## POST /storage/credentials

Generate temporary S3-compatible credentials for accessing your Lyceum storage bucket.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token or API key authentication
</ParamField>

### Response

<ResponseField name="access_key" type="string">
  Temporary S3 access key
</ResponseField>

<ResponseField name="secret_key" type="string">
  Temporary S3 secret key
</ResponseField>

<ResponseField name="session_token" type="string">
  Session token for temporary credentials
</ResponseField>

<ResponseField name="endpoint" type="string">
  S3-compatible endpoint URL
</ResponseField>

<ResponseField name="bucket_name" type="string">
  Your personal storage bucket name
</ResponseField>

<ResponseField name="expires_at" type="string">
  Credential expiration timestamp (12 hours)
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.lyceum.cloud/api/v2/external/storage/credentials \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python
import requests

response = requests.post(
    "https://api.lyceum.cloud/api/v2/external/storage/credentials",
    headers={"Authorization": "Bearer YOUR_TOKEN"}
)
```

```python S3 Client
import boto3

# Get credentials from API
creds = get_storage_credentials()

# Create S3 client
s3 = boto3.client(
    's3',
    endpoint_url=creds['endpoint'],
    aws_access_key_id=creds['access_key'],
    aws_secret_access_key=creds['secret_key'],
    aws_session_token=creds['session_token']
)

# Use the client
s3.list_objects_v2(Bucket=creds['bucket_name'])
```

</RequestExample>
