## POST /auth/api-keys

Create a new API key for the authenticated user.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token authentication
</ParamField>

### Body

<ParamField body="key_name" type="string" required>
  A descriptive name for the API key
</ParamField>

<ParamField body="expires_at" type="datetime">
  Optional expiration date for the API key
</ParamField>

### Response

<ResponseField name="api_key" type="string">
  The generated API key (only shown once)
</ResponseField>

<ResponseField name="api_key_id" type="string">
  Unique identifier for the API key
</ResponseField>

<ResponseField name="key_name" type="string">
  The name assigned to the API key
</ResponseField>

<ResponseField name="key_prefix" type="string">
  The first few characters of the key for identification
</ResponseField>

<ResponseField name="created_at" type="string">
  Creation timestamp
</ResponseField>

<ResponseField name="expires_at" type="string">
  Expiration timestamp if set
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.lyceum.cloud/api/v2/external/auth/api-keys \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "key_name": "Production API Key",
    "expires_at": "2025-01-01T00:00:00Z"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.lyceum.cloud/api/v2/external/auth/api-keys",
    headers={"Authorization": "Bearer YOUR_TOKEN"},
    json={
        "key_name": "Production API Key",
        "expires_at": "2025-01-01T00:00:00Z"
    }
)
```

</RequestExample>

<ResponseExample>

```json 200
{
  "api_key": "lyceum_sk_live_abc123xyz789",
  "api_key_id": "key_123456",
  "key_name": "Production API Key",
  "key_prefix": "lyceum_sk_live_abc",
  "created_at": "2024-01-01T00:00:00Z",
  "expires_at": "2025-01-01T00:00:00Z"
}
```

</ResponseExample>

<Warning>
  Store the API key securely. It will only be shown once and cannot be retrieved later.
</Warning>
